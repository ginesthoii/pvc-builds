# pvc-systems

PVC-based modular systems for water, irrigation, livestock, sensors, and utility infrastructure.

This repository documents low-cost, adaptable, and repairable designs using pressure-rated and gravity-fed PVC components. Builds prioritize modularity, field repair, and reuse over permanence, making them suitable for farms, gardens, workshops, and evolving setups.

This is a systems repository, not a project dump. Patterns are documented once and reused across contexts.
